=========================================
DOKUMENTASI APLIKASI ESTMORE (V1.0)
=========================================

STRUKTUR FOLDER:
1. /css      : Menyimpan file style sheet (CSS) untuk tampilan.
2. /image    : Menyimpan aset gambar produk.
3. /database : Menampilkan script database.

CARA MENJALANKAN:
1. Pastikan XAMPP (Apache & MySQL) sudah aktif.
2. Import database di phpMyAdmin.
3. Buka browser dan akses localhost/estmore.

FILE UTAMA:
- pesanananda.php : Halaman untuk melihat riwayat order.
- koneksi.php     : File konfigurasi ke database.
